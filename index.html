How to Play Blockbuster Puzzle

1

Match Same Blocks

Click or tap on adjacent blocks of the same type to select them. You need to match at least 3 identical blocks in a straight line (horizontal or vertical).

2

Create Combos

The more blocks you match at once, the bigger your combo! Matching 5+ blocks creates special explosive blocks that clear more of the board when used.

3

Complete Objectives

Each level has specific objectives like reaching a target score or clearing certain blocks. Watch the timer and plan your moves strategically!

2

Making Your First Match

Basic Matching Rules

Minimum Match: 3 Blocks

Connect at least 3 identical blocks in a straight line (horizontal or vertical) to make them disappear and score points.

No Diagonal Matches

Blocks must be connected horizontally or vertically - diagonal connections don't count as valid matches.

Tap to Select

Simply tap on one block in a group of 3+ matching blocks to select and clear the entire group.

Scoring System

3 Blocks: 100 points
4 Blocks: 200 points + Special Block
5 Blocks: 500 points + Bomb Block
6+ Blocks: 1000+ points + Rainbow Block

Pro Tip: Larger matches give exponentially more points, so always look for the biggest possible match before making your move!
